# my_pet-projectSQL
# Миграция данных в PostgreSQL

## Цель
Перенос данных из Google Sheets в реляционную БД

## Стек
- СУБД - PostgreSQL
- ОС - Ubuntu (VM)
- клиент для работы с БД - DBeaver

## Структура базы данных
Таблица: "points" (точки продаж)

## Статус
- [+] Установка PostgreSQL
- [+] Создание таблицы
- [+] Загрузка данных
- [+] Написание запросов
## Статистика
- Загружено реальных данных: 280 точек
- Точек имеющих ip-адреса: 92
- Точек имеющих номер AD: 82
- Точек имеющих и ip и AD: 17

## Особенности 
- Все UPID нормализованы до 8 знаков с ведущими нулями
- Триггер автоматически дополняет UPID нулями при вставке
- Ограничение CHECK на длину UPID
- Индекс по id для ускорения сортировки
